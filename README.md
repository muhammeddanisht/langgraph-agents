# LangGraph Agents 🤖

Building production-grade AI agents with LangGraph — 
from basic graphs to multi-agent systems with memory and MCP.

## Versions

| Notebook | What it builds | New concepts |
|----------|---------------|--------------|
| v1_basics | Basic graph — State, Nodes, Edges | Graph structure |
| v2_llm_node | Groq LLM + MemorySaver + chat history | Memory, MessagesState |

## Skills demonstrated
`LangGraph` `LangChain` `Groq` `LLaMA 3.1` `AI Agents` `Python`

## Stack
- LangGraph >= 0.3
- LangChain Groq
- Groq API (LLaMA 3.1 8b-instant)
- Python 3.10+

## How to run
1. Open notebook in Google Colab
2. Add `GROQ_API_KEY` to Colab Secrets
3. Run all cells
